1. paragraph tag

    A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

  2. pre tag

    Text in a pre element is displayed in a fixed-width font, and the text preserves both spaces and line breaks. The text will be displayed exactly as written in the HTML source code.

  3. anchor tag

    The anchor HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

  4. span tag

    The span tag is an inline container used to mark up a part of a text, or a part of a document.

  5. div tag

    The div tag is known as the Division tag. The div tag is used in HTML to make divisions of content in the web page like (text, images, header, footer, navigation bar, etc).
